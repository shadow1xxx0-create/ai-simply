Amazon Rekognition is AWS's cloud-based image and video analysis service. It can detect objects, scenes, faces, text, and inappropriate content — all via simple API calls. Used extensively for content moderation, identity verification, workplace safety, and media analysis at massive scale.

## Key Features

- Object and scene detection in images and video
- Face detection, analysis, and recognition
- Text extraction from images (OCR)
- Content moderation (unsafe content detection)
- Celebrity recognition
- Real-time video analysis

## Pricing

- **Free Tier**: 5,000 images/month for first 12 months
- **Pay-as-you-go**: $0.001 per image ($1 per 1,000)
- Video analysis priced per minute

## Best For

Developers and enterprises building scalable image/video analysis applications on AWS — particularly for content moderation, identity verification, and automated media workflows.
