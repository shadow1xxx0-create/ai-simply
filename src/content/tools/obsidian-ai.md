Obsidian is a local-first note-taking app built around Markdown files stored on your own device. It creates a personal knowledge base where every note can link to others — a "second brain" that maps how your ideas connect.

## Key Features

- Bidirectional linking — notes reference each other automatically
- Graph view visualizes your entire knowledge network
- AI Copilot plugin for writing, summarizing, and Q&A on your notes
- Canvas for visual brainstorming and layouts
- 1,000+ community plugins for every workflow
- Works offline, no subscription required for core features

## Pricing

- **Free**: Core app, unlimited notes, local storage
- **Sync ($5/month)**: End-to-end encrypted sync across devices
- **Publish ($10/month)**: Publish notes as a website
- **Catalyst (one-time $25+)**: Early access to insider builds

## Best For

Researchers, writers, developers, and knowledge workers who want a durable, portable note system they fully own.
