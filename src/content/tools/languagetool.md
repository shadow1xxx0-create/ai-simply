LanguageTool is an open-source grammar and spell checker that supports over 30 languages. It catches complex grammar mistakes, style issues, and punctuation errors in English and non-English text. Available as a browser extension, desktop app, and API.

## Key Features
- 30+ language support
- Style and tone suggestions
- Browser and MS Word add-in
- Team / API access on premium
- Picky mode for advanced style checks
