Aider is an open-source AI pair programming tool that runs in your terminal. It connects to Claude or GPT-4 and lets you make coordinated code changes across multiple files by describing what you want in plain English. It automatically commits changes to git with descriptive messages — making it the most powerful CLI-based AI coding tool available.

## Key Features

- Multi-file code editing from natural language
- Works with Claude, GPT-4, and local models
- Automatic git commits with meaningful messages
- Supports 100+ programming languages
- Voice coding mode
- Completely open-source and free (you pay for API)

## Pricing

- **Free**: Open-source (pay only for LLM API usage)

## Best For

Developers who prefer the terminal, want open-source flexibility, and need an AI coding assistant that can make coordinated changes across an entire codebase — without being locked into a proprietary IDE.
