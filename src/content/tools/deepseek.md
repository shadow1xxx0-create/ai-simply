DeepSeek is a Chinese AI lab that released DeepSeek-R1 and DeepSeek-V3 — models that rival OpenAI's best at dramatically lower costs. The web interface is free to use, and the models are open-source, making DeepSeek a major disruptor in the AI industry. Especially strong at coding and mathematical reasoning.

## Key Features

- DeepSeek-R1: step-by-step reasoning mode
- Exceptional coding and math performance
- Open-source models (MIT license)
- Very low API pricing
- 64K context window
- Available via API and web interface

## Pricing

- **Free**: Web chat with generous limits
- **API**: Extremely competitive pricing vs OpenAI

## Best For

Developers, researchers, and cost-conscious teams who need strong coding and reasoning performance without the enterprise price tag.
