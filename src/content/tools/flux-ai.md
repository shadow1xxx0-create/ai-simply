Flux is a family of state-of-the-art text-to-image models developed by Black Forest Labs, founded by the original creators of Stable Diffusion. Flux.1 produces photorealistic, highly detailed images with exceptional prompt adherence — widely considered the best open-weight image model available. It powers many third-party image generation tools and can be run locally or via API.

## Key Features

- Photorealistic image generation with high prompt accuracy
- Multiple model variants — Flux.1 Pro, Dev, and Schnell
- Open-weight Dev and Schnell models for local use
- Available via API through Replicate, fal.ai, and others
- Exceptional at text rendering within images
- Fast generation on Schnell variant

## Pricing

- **Free**: Via open-source Schnell model (self-hosted)
- **API**: Pay-per-image via Replicate (~$0.003–$0.055/image)
- **Flux Pro**: Via third-party platforms with credits

## Best For

Developers and power users who want the highest quality open-weight image generation model, either self-hosted or via API, with photorealistic results and strong prompt following.
