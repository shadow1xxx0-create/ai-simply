Microsoft Copilot is powered by GPT-4o and built directly into Windows 11, Edge browser, Word, Excel, PowerPoint, and Teams. It can help you write documents, analyze spreadsheets, create presentations, and answer questions — right inside the apps you already use.

## Key Features

- Built into Microsoft 365 apps (Word, Excel, PowerPoint, Teams)
- Windows 11 system integration
- Real-time web search via Bing
- Image generation via DALL-E 3
- Designer for creating visual content
- Copilot+ PC features for edge AI

## Pricing

- **Free**: Copilot in Windows and Edge
- **Microsoft 365 Personal ($7/month)**: Copilot in Office apps
- **Microsoft 365 Business ($30/user/month)**: Enterprise Copilot

## Best For

Microsoft 365 users and businesses already in the Microsoft ecosystem who want AI built directly into their workflow.
