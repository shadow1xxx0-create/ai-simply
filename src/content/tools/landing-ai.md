Landing AI, founded by Andrew Ng, specializes in enterprise computer vision for manufacturing and industrial inspection. Its LandingLens platform lets domain experts build, train, and deploy custom visual AI models without needing ML expertise — using a visual interface to label defects and train models that spot quality issues on production lines.

## Key Features

- Visual AI model training without ML expertise
- Defect detection for manufacturing quality control
- Guided labeling with AI-assisted annotation
- Deploy models to edge devices or cloud
- Iteration tools to continuously improve accuracy
- Enterprise security and compliance
