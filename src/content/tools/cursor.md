Cursor is a VS Code fork rebuilt with AI at its core. Unlike Copilot (a plugin), Cursor's AI can understand your entire codebase — letting you ask questions, make multi-file edits, and refactor large amounts of code with natural language instructions. Rapidly becoming the preferred tool for AI-assisted development.

## Key Features

- Ctrl+K — edit any code with natural language
- Codebase chat — ask questions about your entire project
- Composer — make changes across multiple files
- Multi-model support (GPT-4o, Claude, Gemini)
- All VS Code extensions work as-is
- Agent mode for autonomous coding tasks

## Pricing

- **Free (Hobby)**: 2,000 completions, 50 slow requests/month
- **Pro ($20/month)**: Unlimited completions, 500 fast requests
- **Business ($40/user/month)**: Team features, privacy mode

## Best For

Software engineers and developers who want the deepest possible AI integration in their code editor, especially for large codebases and complex refactors.
