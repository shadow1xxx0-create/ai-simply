Bolt by StackBlitz lets you build complete full-stack web applications from a text prompt. It runs entirely in the browser, installs packages, writes code, and deploys — no local setup required. Bolt can build React, Vue, Next.js, and Astro apps with real databases and backend APIs.

## Key Features

- Full-stack app generation from prompts
- In-browser code execution (no setup)
- Package installation and management
- One-click Netlify deployment
- Edit and iterate with follow-up prompts
- Supports React, Vue, Svelte, Next.js, Astro

- **Free**: 150,000 tokens/day
- **Pro ($20/month)**: 10M tokens/month
- **Teams ($25/user/month)**: Team workspace

Non-technical founders, designers, and developers who want to rapidly prototype and ship full-stack web applications without a complex setup.
