Claude is Anthropic's AI assistant, built with safety and helpfulness in mind. It excels at nuanced writing, complex reasoning, and handling very long documents — with a 200K token context window on paid plans. Claude is widely regarded as one of the best AI models for coding, analysis, and thoughtful conversation.

## Key Features

- 200K token context window (Pro/Team)
- Exceptional document analysis and summarization
- Projects — persistent memory across conversations
- Artifacts — interactive outputs like code and documents
- Computer use capability (beta)
- API access for developers

## Pricing

- **Free**: Claude 3.5 Haiku, limited daily usage
- **Pro ($20/month)**: Claude 3.5 Sonnet, higher limits, priority access
- **Team ($30/user/month)**: Shared projects, admin console
- **Enterprise**: Custom pricing and compliance options

## Best For

Writers, researchers, developers, and analysts who need deep reasoning, long documents, and safe, reliable AI outputs.
