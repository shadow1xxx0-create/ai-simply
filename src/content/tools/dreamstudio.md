DreamStudio is the official web app for Stable Diffusion by Stability AI. It gives you access to the latest SD models (SDXL, SD3) through a clean interface with full parameter control — steps, CFG scale, dimensions, negative prompts — without needing to run anything locally. New accounts get 25 free credits.

## Key Features

- Latest Stable Diffusion models (SDXL, SD3)
- Full parameter control (steps, CFG, sampler)
- Inpainting and image-to-image
- Negative prompts for precise control
- Batch generation
- API access included with account

## Pricing

- **Free**: 25 credits on signup (~200 images)
- **Credits ($10 = 1,000 credits)**: Pay-as-you-go

## Best For

Users who want official Stable Diffusion access without local setup, with full technical control over generation parameters and access to the latest models.
